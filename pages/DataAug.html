Data Augmentation

Important step when building datasets because modern machine learning models are very powerful; if they're given datasets that are too small.

Description

Context: The performance of most ML models, and deep learning models in particular, depends on the quality, quantity and relevancy of training data. However, insufficient data is one of the most common challenges in implementing machine learning in the enterprise.

Problem: Smaller dataset can lead powerful models to overfit the dataset. And hence we are in search of a huge dataset to train such a model. But collecting such data can be costly and time-consuming in many cases.

Solution: Data augmentation is a set of techniques to artificially increase the amount of data by generating new data points from existing data. This includes making small changes to data or using deep learning models to generate new data points.

Example: Image recognition and NLP models generally use data augmentation methods. Also, the medical imaging domain utilizes data augmentation to apply transformations on images and create diversity into the datasets.
